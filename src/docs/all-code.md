
# Nova ERP - الموسوعة البرمجية والتوثيقية النهائية (النسخة الرقابية الشاملة)

هذا المستند هو المرجع المطلق للنظام، تم تحديثه ليشمل ميزة دعم مواد البناء للسكن الخاص الكويتي.

---

## 🏗️ القسم 23: محرك دعم مواد البناء (Subsidy Engine)

تم تصميم هذا المحرك ليتوافق مع قوانين "التموين الإنشائي" في دولة الكويت، حيث يتم صرف مواد مدعومة للمواطنين لا تدخل في حسابات الموردين الخاصة بالشركة.

### 23.1 - مخزن المشروع الافتراضي (Project Quota)
لكل مشروع من فئة **"سكن خاص مدعوم"**، ينشئ النظام سجل حصص (Quotas) يراقب:
- **Allocated**: الكمية المعتمدة من الوزارة (حديد، أسمنت، إلخ).
- **Received**: ما تم استلامه وتوريده فعلياً للموقع.
- **Consumed**: ما تم استهلاكه في الصب والإنشاء بناءً على تقارير التنفيذ.

### 23.2 - المعالجة المحاسبية الصافية
عند استلام مواد مدعومة، لا ينشئ النظام مديونية للمورد (لأن المالك دفعها عبر بطاقة التموين)، ولكن يتم إثباتها دفترياً:
- (Debit) تكلفة المشروع - مواد مدعومة (بالقيمة السوقية للتقييم).
- (Credit) إيرادات دعم مواد (حساب وسيط لبيان وفر التكاليف).

### 23.3 - قاعدة الخصم في المستخلصات (Net Billing Rule)
يطبق النظام قاعدة **"الفوترة الصافية"**:
1. يتم حساب إجمالي قيمة الأعمال (BOQ Progress).
2. يتم طرح إجمالي قيمة المواد المدعومة التي استلمها الموقع.
3. الناتج هو "صافي المطالبة" التي يلتزم المالك بدفعها للشركة نقداً.

## 💰 القسم 9: السلاسل المالية التلقائية (The Automated Chains)

تمت إضافة سلسلة جديدة:
### 9.4 - سلسلة إثبات المدعوم ← المستخلص
- **الحدث**: توريد حديد تموين للموقع.
- **الأتمتة**: تحديث رصيد الـ Quota ← توليد قيد إثبات وفر ← تقليل صافي المستحق في المستخلص القادم آلياً.
