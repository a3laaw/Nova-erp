# نظرة شاملة على نظام Nova ERP

نظام متكامل صمم خصيصاً للمكاتب الهندسية وشركات المقاولات لإدارة العمليات والمحاسبة والموارد البشرية في منصة واحدة.

### المحاور الرئيسية للنظام:

#### 1. إدارة العملاء والمعاملات
مركز التحكم في كافة الخدمات. كل عميل له ملف يحتوي على معاملاته، عقوده، وسجل تفاعلاته (Timeline).

#### 2. نظام المواعيد والتقويم
تنظيم مواعيد الزيارات المعمارية وحجز القاعات مع نظام ذكي لمنع التعارض وتلوين الزيارات حسب حالتها التعاقدية.

#### 3. المحاسبة والمالية
نظام محاسبي متكامل يتبع معايير IFRS، يشمل القيود، السندات، التسويات البنكية، والقوائم المالية الختامية.

#### 4. إدارة الموارد البشرية (HR)
إدارة شؤون الموظفين، الرواتب، الإجازات، والاستئذانات، مع حاسبة دقيقة لمكافأة نهاية الخدمة وفق قانون العمل الكويتي.

#### 5. المخازن والمشتريات
دورة شراء كاملة (RFQ -> PO -> GRN) مع إدارة المستودعات وتحليل حركة الأصناف وتكاليفها باستخدام الذكاء الاصطناعي.

### الميزات الذكية:
- **المساعد المحاسبي:** تنفيذ العمليات بالأوامر النصية.
- **خبير النظام:** الإجابة على استفسارات المستخدمين حول البيانات.
- **محلل المستندات:** استخراج البيانات من صور عروض الأسعار آلياً.
