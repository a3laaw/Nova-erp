# Nova ERP - نظام الإدارة المتكامل للمكاتب الهندسية وشركات المقاولات

نظام احترافي شامل مصمم خصيصاً للسوق الكويتي، يربط العمليات الهندسية الميدانية بالدورة المستندية والمحاسبية والموارد البشرية في منصة واحدة.

## 🌟 المميزات الرئيسية

### 🏗️ إدارة المقاولات (WBS)
- بناء مراحل عمل شجرية لكل نوع من أنواع المقاولات (هيكل أسود، تشطيب، ترميم).
- إدارة جداول الكميات (BOQ) وربطها بالتكاليف الفعلية.
- أتمتة المستخلصات المالية بناءً على تأكيد الإنجاز الميداني عبر الـ GPS.

### 💰 المحاسبة والمالية (IFRS)
- شجرة حسابات مرنة تدعم مراكز التكلفة والربحية التلقائية لكل مشروع.
- مساعد محاسبي مدعوم بالذكاء الاصطناعي (AI) لتسجيل العمليات بالأوامر الصوتية أو النصية.
- دورة مشتريات ذكية (RFQ -> PO -> GRN) مع تحليل أسعار الموردين.

### 👥 الموارد البشرية (HR & Payroll)
- ملفات موظفين متكاملة تشمل تتبع الإقامات والعقود.
- معالجة رواتب ذكية تدعم ملفات البصمة (Excel) والاستئذانات.
- حاسبة نهاية الخدمة وفقاً لقانون العمل الكويتي (المادة 51).

### 🗓️ المواعيد والتقويم
- تقويم مزدوج (للمعماريين وللقاعات) يمنع تعارض المواعيد فورياً.
- تتبع حالة الزيارات وتلوينها آلياً حسب الحالة التعاقدية للعميل.

## 🛠️ التقنيات المستخدمة
- **Framework**: Next.js 14 (App Router).
- **Database & Auth**: Firebase (Firestore & Authentication).
- **Styling**: Tailwind CSS & ShadCN UI.
- **AI Engine**: Google Gemini AI.
