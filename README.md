# Nova ERP - النظام المتكامل للمكاتب الهندسية وشركات المقاولات

نظام متطور وشامل مصمم خصيصاً لإدارة العمليات الهندسية، المحاسبية، والموارد البشرية في منصة واحدة متكاملة.

## 🚀 المميزات الرئيسية

### 🏗️ إدارة المقاولات والمشاريع
- **نظام WBS شجري**: بناء مراحل عمل لامتناهية لكل نوع من أنواع المقاولات.
- **جداول كميات (BOQ)**: إدارة الحصر والتسعير والربحية لكل بند.
- **أتمتة المستخلصات**: توليد مطالبات مالية آلياً فور إنجاز المراحل الميدانية.
- **زيارات ميدانية**: نظام تأكيد الحضور عبر الـ GPS وتوثيق الموقع بالصور.

### 💰 المحاسبة والمالية
- **شجرة حسابات احترافية**: مطابقة لمعايير IFRS مع إنشاء حسابات تلقائية للعملاء.
- **مساعد محاسبي AI**: تسجيل العمليات بالأوامر الصوتية أو النصية باللغة العربية.
- **تقارير ختامية**: قائمة دخل، مركز مالي، وتدفقات نقدية لحظية.
- **تسويات بنكية ذكية**: مطابقة كشوف الحسابات بالذكاء الاصطناعي.

### 👥 الموارد البشرية (HR)
- **رواتب قانونية**: متوافقة تماماً مع قانون العمل الكويتي.
- **حاسبة نهاية الخدمة**: حساب دقيق للمستحقات وبدل الإجازات.
- **نظام إجازات هجين**: تقديم إلكتروني مع طباعة نماذج رسمية للتوقيع اليدوي.

## 🛠️ التقنيات المستخدمة
- **Next.js 14**: لضمان أسرع أداء للواجهات.
- **Firebase**: كقاعدة بيانات سحابية لحظية ومؤمنة.
- **Genkit & Gemini**: لمحركات الذكاء الاصطناعي في النظام.
- **Tailwind CSS & ShadCN**: لتصميم عصري واحترافي.
