# Nova ERP - نظام إدارة الأعمال المتكامل

نظام متطور مصمم خصيصاً للمكاتب الهندسية وشركات المقاولات لإدارة العمليات والمحاسبة والموارد البشرية في منصة واحدة.

## المميزات الرئيسية
- **إدارة العملاء (CRM):** متابعة الملفات، المعاملات، والزيارات.
- **المحاسبة والمالية:** شجرة حسابات، قيود، سندات، وقوائم مالية (IFRS).
- **الموارد البشرية:** رواتب، إجازات، ونهاية خدمة (قانون الكويت).
- **المخازن والمشاريع:** تتبع المواد، أوامر الشراء، وجداول الكميات (BOQ).

## التقنيات
- **Frontend:** Next.js, TypeScript, Tailwind CSS, ShadCN UI.
- **Backend:** Firebase (Firestore, Auth, Storage).
