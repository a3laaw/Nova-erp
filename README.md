
# Nova ERP - نظام الإدارة المتكامل للمكاتب الهندسية وشركات المقاولات

نظام احترافي شامل مصمم خصيصاً للسوق الكويتي، يربط العمليات الهندسية الميدانية بالدورة المستندية والمحاسبية والموارد البشرية في منصة واحدة.

**Nova ERP - Built with Firebase Studio - March 2026**

## 🌟 المميزات السيادية

### 🏗️ إدارة المقاولات (WBS & GPS)
- بناء مراحل عمل شجرية وتتبع الإنجاز الميداني عبر الـ GPS.
- أتمتة المستخلصات المالية بناءً على الواقع الميداني.

### 💰 المحاسبة والمالية (IFRS)
- شجرة حسابات مرنة تدعم مراكز التكلفة والربحية التلقائية.
- معالجة ذكية للعمولات البنكية ومرتجعات الموردين.

### 👥 الموارد البشرية والخدمة الذاتية (HR)
- رادار ذكي لمنع تداخل الإجازات والاستئذانات.
- شفافية كاملة للموظف لرؤية ردود الإدارة وأسباب الرفض.
